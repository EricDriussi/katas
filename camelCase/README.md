# CamelCase Kata

Write a program that converts text to CamelCase.
The input text can have hyphen, underscore and/or space separated words.
The words in the input text may or may not be capitalized.

## Examples

- "" => ""
- "Foo" => "Foo"
- "Foo Bar" => "FooBar"
- "Foo_Bar-Foo" => "FooBarFoo"
- "foo" => "Foo"
- "foo_bar foo-bar" => "FooBarFooBar"
